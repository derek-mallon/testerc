Detailed Description

Header only simply unit testing framework.

dsaslfkds Usage: 

  int test_var;
  ENVIROMENT_SETUP{ //Run before each unit test.
      test_var = 5;
  }

  ENVIROMENT_CLEANUP{ //Run after each unit test.
  }

  TESTS
      UNIT_TEST_START("test if test_var is 5")
          ASSERT(test_var == 5);
      UNIT_TEST_END
      UNIT_TEST_START("test if test_var++ is 6")
          test_var++;
          ASSERT(test_var == 6);
      UNIT_TEST_END
  END_TESTS
